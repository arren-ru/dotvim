= ~/.vim/

My settings for Vim. See comments in link:vimrc[] for list of supported
features (in Russian).

== Install

Remove `~/.vimrc` and `~/.vim/`, then:

```
git clone --recursive https://github.com/powerman/dotvim ~/.vim
```

== Update

```
cd ~/.vim
git pull
git submodule sync
git submodule update
```
